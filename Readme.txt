# Breast Cancer FNA Image Classification

## Requirements
- Python 3.7+
- TensorFlow 2.x
- OpenCV
- NumPy
- Matplotlib
- scikit-learn

## Setup
1. Install required packages:
   pip install tensorflow opencv-python numpy matplotlib scikit-learn

2. Ensure proper directory structure with Dataset2 folder containing FNA and test folders

3. Run the script:
   python Project5.py

## Output
- training_history.png: Plot of training/validation accuracy and loss
- predictions.txt: Predictions for test images
- breast_cancer_model.h5: Saved model file
